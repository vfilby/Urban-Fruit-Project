# Urban Fruit Project

A community platform for cataloging perennial food-bearing plants on public land.
Map trees, bushes, and vines — things you can find once and return to year after year.

## Project Philosophy

**Specification-driven development.** The spec and acceptance criteria are the
durable artifacts. Code is regenerable from the spec at any time.

- `spec/` — Project specification (the source of truth)
- `archive/` — Original 2012 Rails codebase (reference only)
